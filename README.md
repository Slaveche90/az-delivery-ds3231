# Real Time Clock DS3231

Python sripts for using DS3231 with Raspberry Pi

# Tested

Tested with Raspberry Pi 3 model B+ and RTC DS3231 (https://www.az-delivery.de/products/ds3231-real-time-clock?_pos=1&_sid=5d0475c2a&_ss=r)

# Usage

rtc_lib.py contains all functions needed for controlling the RTC.  
rtc_example.py is an example script, showing how to use the rtc_lib.py

# Reference

The rtc_lib.py script is modified script (https://github.com/switchdoclabs/RTC_SDL_DS3231/blob/master/SDL_DS3231.py). Script is modified to be used with Python 3 and to get better output.
